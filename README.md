## About The Project

This is the official digital platform for **Water and Wisdom** — a high-impact, visually stunning website built to deliver educational resources, interactive tools for water tracking, community forums, expert insights, and partnership opportunities.

Designed with precision to reflect the project's harmonious blend of tradition and innovation.

**GitHub Repository**: [github.com/biospaceafrica-art/waterandwisdom](https://github.com/biospaceafrica-art/waterandwisdom)

## ✨ How to Run Locally

```bash
# Clone the repository
git clone https://github.com/biospaceafrica-art/waterandwisdom.git

# Navigate into the project directory
cd waterandwisdom

# Install dependencies
npm install

# Start the development server
npm run dev
